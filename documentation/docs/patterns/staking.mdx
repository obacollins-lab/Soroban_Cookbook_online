## Overview

This pattern implements a **token staking contract** with pro-rata reward distribution over fixed-length epochs.  Users stake any positive amount at any time; rewards accrue each epoch proportional to their share of the total stake and can be claimed at any time without unstaking.

Key design decisions:

- **Epoch-based accounting** — rewards are distributed per complete epoch, defined as a fixed number of ledgers.
- **Lazy settlement** — each stake record stores a `reward_debt` (last-claim epoch), so pending rewards are computed on-demand without iterating all stakers.
- **No external token contract** — this example is self-contained and tracks balances internally, making it easy to study in isolation.  Wire a real SEP-41 / SAC token interface for production use.

<PatternSection id="prerequisites" title="Prerequisites">

- Rust stable and `cargo`
- Soroban CLI installed ([setup guide](/docs/getting-started/setup))
- Familiarity with `soroban_sdk` persistent storage and `require_auth`
- Understanding of the [token-transfer](/docs/patterns/overview) pattern (issue #81)

</PatternSection>

<PatternSection id="implementation" title="Implementation">

<PatternCallout variant="info" title="Epoch formula">

`epoch = (current_ledger − start_ledger) / epoch_len`

Rewards for a user are: `epochs_elapsed × reward_per_epoch × user_stake / total_staked`.

</PatternCallout>

```rust
#![no_std]

use soroban_sdk::{contract, contracterror, contractimpl, contracttype, Address, Env};

#[contracttype]
#[derive(Clone)]
pub enum DataKey {
    Admin,
    EpochLen,
    RewardPerEpoch,
    StartLedger,
    TotalStaked,
    Stake(Address),
}

#[contracttype]
#[derive(Clone, Debug, Eq, PartialEq)]
pub struct StakeInfo {
    pub amount: i128,
    /// Epoch index of the last claim (or first stake).
    pub reward_debt: u32,
}

#[contracterror]
#[derive(Copy, Clone, Debug, Eq, PartialEq, PartialOrd, Ord)]
#[repr(u32)]
pub enum Error {
    AlreadyInitialized  = 1,
    NotInitialized      = 2,
    Unauthorized        = 3,
    InvalidAmount       = 4,
    NothingStaked       = 5,
    InvalidEpochLen     = 6,
    InvalidReward       = 7,
}

#[contract]
pub struct Staking;

#[contractimpl]
impl Staking {
    /// Initialise the staking contract.
    pub fn initialize(
        env: Env,
        admin: Address,
        epoch_len: u32,
        reward_per_epoch: i128,
    ) -> Result<(), Error> {
        if epoch_len == 0 { return Err(Error::InvalidEpochLen); }
        if reward_per_epoch <= 0 { return Err(Error::InvalidReward); }
        if env.storage().instance().has(&DataKey::Admin) {
            return Err(Error::AlreadyInitialized);
        }
        admin.require_auth();
        let s = env.storage().instance();
        s.set(&DataKey::Admin,          &admin);
        s.set(&DataKey::EpochLen,        &epoch_len);
        s.set(&DataKey::RewardPerEpoch,  &reward_per_epoch);
        s.set(&DataKey::StartLedger,     &env.ledger().sequence());
        s.set(&DataKey::TotalStaked,     &0_i128);
        Ok(())
    }

    /// Stake `amount` tokens on behalf of `staker`.
    pub fn stake(env: Env, staker: Address, amount: i128) -> Result<(), Error> {
        if amount <= 0 { return Err(Error::InvalidAmount); }
        staker.require_auth();
        Self::require_initialized(&env)?;

        let epoch = Self::current_epoch(&env);
        let mut info = Self::load_stake(&env, &staker);
        info.reward_debt = epoch;
        info.amount = info.amount.checked_add(amount).expect("overflow");
        Self::save_stake(&env, &staker, &info);

        let total: i128 = env.storage().instance()
            .get(&DataKey::TotalStaked).unwrap_or(0);
        env.storage().instance().set(&DataKey::TotalStaked, &(total + amount));
        Ok(())
    }

    /// Unstake `amount` tokens; returns accrued rewards.
    pub fn unstake(env: Env, staker: Address, amount: i128) -> Result<i128, Error> {
        if amount <= 0 { return Err(Error::InvalidAmount); }
        staker.require_auth();
        Self::require_initialized(&env)?;

        let epoch = Self::current_epoch(&env);
        let mut info = Self::load_stake(&env, &staker);
        if info.amount == 0 { return Err(Error::NothingStaked); }
        if amount > info.amount { return Err(Error::InvalidAmount); }

        let pending = Self::pending_rewards(&env, &info, epoch);
        info.reward_debt = epoch;
        info.amount -= amount;
        Self::save_stake(&env, &staker, &info);

        let total: i128 = env.storage().instance()
            .get(&DataKey::TotalStaked).unwrap_or(0);
        env.storage().instance().set(&DataKey::TotalStaked, &(total - amount));
        Ok(pending)
    }

    /// Claim pending rewards without unstaking.
    pub fn claim(env: Env, staker: Address) -> Result<i128, Error> {
        staker.require_auth();
        Self::require_initialized(&env)?;

        let epoch = Self::current_epoch(&env);
        let mut info = Self::load_stake(&env, &staker);
        if info.amount == 0 { return Err(Error::NothingStaked); }

        let pending = Self::pending_rewards(&env, &info, epoch);
        info.reward_debt = epoch;
        Self::save_stake(&env, &staker, &info);
        Ok(pending)
    }

    // ── view functions ────────────────────────────────────────────────

    pub fn staked_balance(env: Env, staker: Address) -> i128 {
        Self::load_stake(&env, &staker).amount
    }

    pub fn pending_reward(env: Env, staker: Address) -> i128 {
        if env.storage().instance().has(&DataKey::StartLedger) {
            let epoch = Self::current_epoch(&env);
            let info  = Self::load_stake(&env, &staker);
            Self::pending_rewards(&env, &info, epoch)
        } else { 0 }
    }

    pub fn total_staked(env: Env) -> i128 {
        env.storage().instance().get(&DataKey::TotalStaked).unwrap_or(0)
    }

    pub fn current_epoch(env: &Env) -> u32 {
        let start: u32 = env.storage().instance()
            .get(&DataKey::StartLedger).unwrap_or(0);
        let len: u32   = env.storage().instance()
            .get(&DataKey::EpochLen).unwrap_or(1);
        env.ledger().sequence().saturating_sub(start) / len
    }

    // ── admin ─────────────────────────────────────────────────────────

    pub fn set_reward(env: Env, reward_per_epoch: i128) -> Result<(), Error> {
        if reward_per_epoch <= 0 { return Err(Error::InvalidReward); }
        let admin = Self::require_admin(&env)?;
        admin.require_auth();
        env.storage().instance().set(&DataKey::RewardPerEpoch, &reward_per_epoch);
        Ok(())
    }

    // ── helpers ───────────────────────────────────────────────────────

    fn require_initialized(env: &Env) -> Result<(), Error> {
        if !env.storage().instance().has(&DataKey::Admin) {
            return Err(Error::NotInitialized);
        }
        Ok(())
    }

    fn require_admin(env: &Env) -> Result<Address, Error> {
        env.storage().instance().get(&DataKey::Admin).ok_or(Error::NotInitialized)
    }

    fn load_stake(env: &Env, staker: &Address) -> StakeInfo {
        env.storage().persistent()
            .get(&DataKey::Stake(staker.clone()))
            .unwrap_or(StakeInfo { amount: 0, reward_debt: 0 })
    }

    fn save_stake(env: &Env, staker: &Address, info: &StakeInfo) {
        env.storage().persistent().set(&DataKey::Stake(staker.clone()), info);
    }

    fn pending_rewards(env: &Env, info: &StakeInfo, current_epoch: u32) -> i128 {
        if info.amount == 0 || current_epoch <= info.reward_debt { return 0; }
        let elapsed       = (current_epoch - info.reward_debt) as i128;
        let reward: i128  = env.storage().instance()
            .get(&DataKey::RewardPerEpoch).unwrap_or(0);
        let total: i128   = env.storage().instance()
            .get(&DataKey::TotalStaked).unwrap_or(1);
        if total == 0 { return 0; }
        elapsed * reward * info.amount / total
    }
}
```

```rust
#[cfg(test)]
mod tests {
    use super::*;
    use soroban_sdk::{testutils::Address as _, Env};

    fn setup() -> (Env, Address, StakingClient<'static>) {
        let env = Env::default();
        env.mock_all_auths();
        let contract_id = env.register(Staking, ());
        let client = StakingClient::new(&env, &contract_id);
        let admin = Address::generate(&env);
        // epoch_len = 10 ledgers, reward_per_epoch = 1000
        client.initialize(&admin, &10, &1000);
        (env, admin, client)
    }

    #[test]
    fn test_stake_increases_balance_and_total() {
        let (env, _, client) = setup();
        let alice = Address::generate(&env);
        client.stake(&alice, &500);
        assert_eq!(client.staked_balance(&alice), 500);
        assert_eq!(client.total_staked(), 500);
    }

    #[test]
    fn test_reward_after_one_epoch() {
        let (env, _, client) = setup();
        let alice = Address::generate(&env);
        client.stake(&alice, &1000);
        env.ledger().set_sequence_number(10); // 1 full epoch
        assert_eq!(client.pending_reward(&alice), 1000);
    }

    #[test]
    fn test_pro_rata_reward_two_stakers() {
        let (env, _, client) = setup();
        let alice = Address::generate(&env);
        let bob   = Address::generate(&env);
        client.stake(&alice, &750);
        client.stake(&bob,   &250);
        env.ledger().set_sequence_number(10);
        assert_eq!(client.pending_reward(&alice), 750);
        assert_eq!(client.pending_reward(&bob),   250);
    }

    #[test]
    fn test_claim_resets_reward_debt() {
        let (env, _, client) = setup();
        let alice = Address::generate(&env);
        client.stake(&alice, &1000);
        env.ledger().set_sequence_number(10);
        let claimed = client.claim(&alice);
        assert_eq!(claimed, 1000);
        assert_eq!(client.pending_reward(&alice), 0);
    }

    #[test]
    fn test_unstake_returns_accrued_reward() {
        let (env, _, client) = setup();
        let alice = Address::generate(&env);
        client.stake(&alice, &1000);
        env.ledger().set_sequence_number(20); // 2 epochs
        let reward = client.unstake(&alice, &1000);
        assert_eq!(reward, 2000);
        assert_eq!(client.staked_balance(&alice), 0);
    }

    #[test]
    fn test_unstake_more_than_staked_is_rejected() {
        let (env, _, client) = setup();
        let alice = Address::generate(&env);
        client.stake(&alice, &100);
        let result = client.try_unstake(&alice, &200);
        assert_eq!(result, Err(Ok(Error::InvalidAmount)));
    }

    #[test]
    fn test_double_initialize_is_rejected() {
        let (env, admin, client) = setup();
        let result = client.try_initialize(&admin, &10, &1000);
        assert_eq!(result, Err(Ok(Error::AlreadyInitialized)));
    }
}
```

<PatternSection id="security" title="Security considerations">

- **Authorization** — every state-mutating function calls `staker.require_auth()` or `admin.require_auth()` before touching storage.
- **Integer overflow** — stake accumulation uses `checked_add` to prevent silent wrapping; `overflow-checks = true` in the release profile catches arithmetic panics.
- **Division by zero** — `pending_rewards` guards against a zero `total_staked` before dividing.
- **Epoch manipulation** — epoch index is derived from `env.ledger().sequence()`, which is set by the network and cannot be spoofed by a caller.
- **No external token calls** — the example avoids reentrancy by not calling into other contracts.  In production, transfer tokens *after* updating state (checks-effects-interactions).

<PatternCallout variant="warning" title="Production checklist">

Before mainnet: replace the internal balance tracking with a real SEP-41 / SAC token interface, add storage TTL bumps for persistent entries, and audit the reward math for precision loss on large epoch counts.

</PatternCallout>
